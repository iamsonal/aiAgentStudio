# AI Agent Studio Docs (Astro + Starlight)

This is the new documentation site using Astro Starlight with an enterprise control-plane visual style.

## Run locally

```bash
cd docs-starlight
npm install
npm run dev
```

## Build

```bash
cd docs-starlight
npm run build
```

## Notes

- Active docs content lives in `src/content/docs/`.
- Keep `base` in `astro.config.mjs` aligned with your GitHub Pages path.
